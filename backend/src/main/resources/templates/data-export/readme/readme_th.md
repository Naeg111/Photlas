## โครงสร้างไฟล์

| ไฟล์ / โฟลเดอร์              | คำอธิบาย |
|------------------------------|----------|
| `README.md`                  | ไฟล์นี้ (โครงสร้างไฟล์และคำอธิบายฟิลด์) |
| `user.json`                  | ข้อมูลโปรไฟล์ (ชื่อแสดงผล อีเมล ภาษา ฯลฯ) |
| `profile_image.*`            | รูปโปรไฟล์ (ถ้ามี) |
| `photos.json`                | ข้อมูลเมตาของรูปภาพที่อัปโหลด |
| `photos/`                    | ไฟล์รูปต้นฉบับ (`{photoId}.{นามสกุล}`) |
| `favorites.json`             | รูปภาพที่คุณกดถูกใจ |
| `sns_links.json`             | ลิงก์โซเชียลในโปรไฟล์ |
| `oauth.json`                 | บัญชี OAuth ที่เชื่อมโยง (Google / LINE) |
| `reports.json`               | ประวัติการรายงานของคุณ |
| `sanctions.json`             | ประวัติการลงโทษบัญชี (ไม่รวมข้อความอิสระจากผู้ดูแลระบบ) |
| `violations.json`            | ประวัติการละเมิด (ไม่รวมข้อความอิสระจากผู้ดูแลระบบ) |
| `location_suggestions.json`  | คำแนะนำการแก้ไขตำแหน่งของคุณ |
| `spots.json`                 | จุดถ่ายภาพที่คุณสร้าง |
| `errors.json`                | บันทึกข้อผิดพลาด (ว่างเปล่าหากไม่มี) |
| `_complete.flag`             | เครื่องหมายความสมบูรณ์ของไฟล์เก็บถาวร (มีอยู่ = สมบูรณ์) |

## คำอธิบายฟิลด์ที่สำคัญ

### `photos.json` ฟิลด์ `moderationStatus`
- `1001` = PENDING_REVIEW (รอตรวจสอบ)
- `1002` = PUBLISHED (เผยแพร่แล้ว)
- `1003` = QUARANTINED (กักไว้)
- `1004` = REMOVED (ลบแล้ว) — ไม่มีไฟล์รูปภาพต้นฉบับ

### รูปแบบวันเวลา
- วันเวลาในไฟล์ JSON ทั้งหมดอยู่ในรูปแบบ ISO 8601 **UTC** (ลงท้ายด้วย `Z` เช่น `2026-03-10T05:30:00Z`)
- เฉพาะไฟล์นี้และอีเมลแจ้งเตือนเท่านั้นที่แสดงเวลาตามเขตเวลาท้องถิ่นของภาษาที่คุณลงทะเบียน

## ข้อควรระวัง

- ไฟล์เก็บถาวรนี้มีข้อมูลส่วนบุคคล (อีเมล พิกัดการถ่ายภาพ ข้อมูลอุปกรณ์ ฯลฯ) โปรดใช้ความระมัดระวังเมื่อแชร์กับผู้อื่น
- หากไม่มี `_complete.flag` ในไฟล์ การดาวน์โหลดอาจถูกขัดจังหวะ โปรดร้องขอใหม่อีกครั้ง
- รูปภาพสถานะ REMOVED จะมีเฉพาะข้อมูลเมตา ไม่มีการแจกจ่ายไฟล์รูปภาพ
- หากมีข้อสงสัยหรือต้องการเปิดเผยข้อมูลเพิ่มเติม โปรดติดต่อ support@photlas.jp
