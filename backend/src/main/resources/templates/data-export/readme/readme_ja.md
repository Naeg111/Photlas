## ファイル構成

| ファイル / ディレクトリ | 内容 |
|--------------------------|------|
| `README.md`              | 本ファイル（ZIP 構成と各フィールドの説明） |
| `user.json`              | プロフィール情報（表示名・メール・登録言語等） |
| `profile_image.*`        | プロフィール画像（設定されている場合のみ） |
| `photos.json`            | 投稿写真のメタデータ一覧 |
| `photos/`                | 投稿写真の元画像（`{photoId}.{拡張子}`） |
| `favorites.json`         | お気に入り写真の一覧 |
| `sns_links.json`         | プロフィールに登録された SNS リンク |
| `oauth.json`             | OAuth 連携先（Google / LINE 等） |
| `reports.json`           | あなたが行った通報の履歴 |
| `sanctions.json`         | アカウント制裁履歴（管理者の自由記述は除外） |
| `violations.json`        | 違反履歴（管理者の自由記述は除外） |
| `location_suggestions.json` | あなたが行った位置情報修正提案 |
| `spots.json`             | あなたが作成した撮影スポット |
| `errors.json`            | 画像ダウンロード失敗等のエラー記録（無ければ空配列） |
| `_complete.flag`         | ZIP の完全性マーカー（このファイルが存在すれば完全） |

## 主要フィールドの意味

### `photos.json` の `moderationStatus`
- `1001` = PENDING_REVIEW（審査待ち）
- `1002` = PUBLISHED（公開中）
- `1003` = QUARANTINED（隔離中）
- `1004` = REMOVED（削除済み）— 画像ファイル本体は ZIP に含まれません

### 日時の表記
- すべての JSON ファイル内の日時は **UTC** の ISO 8601 形式（末尾 `Z`、例: `2026-03-10T05:30:00Z`）です
- このファイルとメール通知の日時のみ、登録言語に応じた現地時間で表示されます

## 重要な注意事項

- このアーカイブには個人情報（メールアドレス・撮影位置情報・端末情報等）が含まれます。第三者と共有する際は十分にご注意ください。
- `_complete.flag` が ZIP 内に存在しない場合、ダウンロードが途中で中断された可能性があります。再度ダウンロードしてください。
- REMOVED 状態の写真はメタデータのみが含まれ、画像本体は再配布されません。
- データに不備がある、追加で開示してほしい情報があるなどの場合は support@photlas.jp までお問い合わせください。
